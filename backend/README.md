# HireMind Backend

ASP.NET backend structure for the AI Recruitment Platform.

## Main Areas
- `src/HireMind.Api` - API layer, controllers, middleware, request/response models
- `src/HireMind.Application` - use cases, services, DTOs, validation, interfaces
- `src/HireMind.Domain` - entities, value objects, enums, domain contracts
- `src/HireMind.Infrastructure` - persistence, repositories, identity, external integrations
- `db` - SQL schema, migrations, seeds, scripts
- `tests` - automated tests for backend layers
